# 1. ![Title](./README_Image/Title.png) 소개

  
  ### **쉽게 하는 간편이사(Umzip🚚🏠) 소규모 이사**
    
  🚛 Naming : 움(Um)직이는 집(zip)처럼 손쉽게 이사를 한다는 의로 움집(Umzip) 탄생.
  
   늘어나는 1인가구에 따라 소규모 이사의 수요가 증가 이게 맞춰 
  
  용달, 청소, 중고 거래, 도움과 커뮤니티 등 원활한 이사를 지원하는 서비스.
